# PNG-Steganograph

## Quick Start

```console
$ yarn --frozen-lockfile
$ yarn run start
```

## Global usage

### Build:
```console
$ yarn run build
```

### Install package globally:
```console
$ npm install -g
```

### Use the command:
```console
$ steg
```
