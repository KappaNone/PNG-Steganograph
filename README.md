# PNG-Steganograph

## Quick Start

```console
$ yarn --frozen-lockfile install
$ yarn run start
```

## Global usage

### Install `tsx` globally:
    ```console
    $ npm install tsx -g
    ```

### Install dependencies globally:
    ```console
    $ npm install -g
    ```

### To run the script, use the command:
    ```console
    $ steg
    ```
